Ajouter un Département
Informations du département
Règles de création
  • Le nom, le code et le type sont obligatoires
  • Le code doit être unique dans le système (ex: RH, IT, FIN)
  • L'email doit être une adresse valide
  • Le type Opérationnel permet d'associer le département à des projets
  • Le type Support est transversal et ne peut pas être lié à des projets
  • Le département sera actif par défaut après sa création
